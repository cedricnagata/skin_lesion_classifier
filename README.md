# skin_lesion_classifier
Machine Learning models that use ResNet50 CNNs with additional training and image processing to diagnose a skin lesion as a nevus, melanoma, or other and as benign or malignant. Trained on 44,033 images from the ISIC public archive.

## Model Creation and Training

### Tech Stack
- **Python** for data organization, image processing, model creation, training, and evaluation
- **TensorFlow** for the machine learning models
- **Flask** for backend API (serving predictions)
- **Google Colab** for computing environment
